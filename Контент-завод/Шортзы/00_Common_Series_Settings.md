# Tokenova — серия Shorts/Reels (общие параметры)

## Формат
- Вертикальное видео: **9:16** (1080×1920)
- Длительность: обычно **30–45 сек**, для “how-to” допустимо **55–65 сек**
- FPS: **30** (или 60 для активного скринкаста)
- Аудио: голос + тихий фон; лёгкие SFX (клики/свайпы)
- Субтитры: **всегда**, 2 строки, крупно

## Визуальный стиль
- Палитра: графит/серый + акцент **фуксия**
- Шрифт: простой гротеск (Inter/Montserrat), жирный для заголовков
- Темп: 1 мысль = 1 сцена, без длинных статичных кадров
- На экране постоянно “якорь”: лицо / интерфейс / код / объект недвижимости

## Шаблон оверлеев
- 0–2 сек: хук (1 короткая фраза)
- 2–20 сек: 2–3 тезиса по 3–5 слов
- Финал: CTA + «ссылка в описании»

## Обязательные ассеты
- Скринкаст: вход в Telegram-бот → TonConnect → список проектов → купить → подтверждение → токены
- Код/скрин: фрагмент **`refund()`** (крупно, подсветка строки)
- B-roll: море/город/объекты/документы (SPV/доля/право)
